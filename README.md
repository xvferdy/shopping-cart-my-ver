## Getting Started
### Prerequisites
- [Node.js](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)


### Installation
```
npm install
```
```
npm start
```
**Deployment**
```
npm install --save gh-pages
```
###### package.json
```JSON
  "homepage": "https://myusername.github.io/shopping-cart-my-ver",
  "scripts": {
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build",
  },
```
```
npm run deploy
```
